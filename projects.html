Project 01 · 2025 · Hackathon

Hunting for Exoplanets with AI

Featured Hackathon

Built an AI model to detect potential exoplanets by analyzing telescope light-curve data. Used deep learning and signal-processing techniques to identify brightness dips caused by orbiting planets. Achieved 86% classification accuracy and built a real-time Streamlit interface for visualization.

  • Analyzed stellar light-curve datasets to detect planetary transit signals
  • Designed a Convolutional Neural Network (CNN) for time-series classification
  • Applied signal processing to clean, normalize and preprocess light data
  • Achieved 86.00% accuracy on exoplanet candidate classification
  • Built a real-time Streamlit UI to visualize detection results interactively
